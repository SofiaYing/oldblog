切图

psd

  • 查看图层：

（1）复杂图层：右键，选中图层名称。

（2）简单图层：可以在右侧点击眼睛图片显示隐藏图层

  • 导出图层：

(1) 在右侧所需图层中，右键，duplicate layer,重新保存document。

(2) 上侧导航栏，image，trim,ok.

(3) 上侧导航栏，image，anves size(画布大小)，改成相应的尺寸.

(4) file export quick export as PNG（旧版ps：save for web,选择保存为png。）

(5) 测量图片直接间距，如果用Ps,可以用选中图片A，alt+command,再选中图片B。

抠图（建议设计师给psd）

（1）image crop,可以对png等进行粗裁剪

（2）(ps如何选择想要的像素/如何使用魔法棒工具)左侧工具栏 魔法棒 magic wand tool,选择想要留下的区域，按一下shift可以再多选择，参考下图。

（3）右键select inverse（反选），点击delete，可以清楚不想要的区域（就变成了透明背景）.

（4）右键deselect(取消选择)。

CSS Sprites (精灵图)雪碧图:把多张图连城一个图,可以将图标合并

查询css sprites generator，可以在线生成，拷贝代码即可。
原理：将多个图片合并成一个图标，所有div的背景都是一整张图片，利用background-position，width,height显示图片不同位置，视觉上只显示了一个图标。

iconfont字体图标 html

  • 字体图标编码：&#xe61…。约定促成的，iconfont一般以e61开始，因为这一段是不表示任何字符的，万一字体没有加载，也不会出现错误，变成其他字。

  • 基础知识补充
    转义/实体:&entityName或&#entityNumber，超过某个数字时，则以&#x开头，;表示结束。
    具体的参照Unicode编码，可以查询在线转换工具。
    空格：&nbsp或&#60 如：1&60;2

  • 如应用iconfont网站上的图标（具体用法可以参照图标下载页面的使用帮助）

iconfont css

iconfont symbol(svg)(优先使用)

  • 可以使用sketch，ps等工具将图片导出为svg，具体网上查询。

  • SVG canvas png 具体网上查询.

  • 这种用法其实是做了一个svg的集合，与上面两种相比具有如下特点：

  1. 支持多色图标了，不再受单色限制。
  2. 通过一些技巧，支持像字体那样，通过font-size,color来调整样式。
  3. 兼容性较差，支持 ie9+,及现代浏览器。
  4. 浏览器渲染svg的性能一般，还不如png。
