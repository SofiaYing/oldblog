2018-05-29

flex（flexible 弹性、灵活）

  1. 块级布局侧重垂直方向、行内布局侧重水平方，向flex布局与方向无关。
  2. flex布局可以实现空间自动分配、元素自动对齐。
  3. 适用于简单的线性布，更复杂的布局如瀑布流等，建议使用js或者grid。
  • flex container属性
  1. flex-direction 方向 : row(行，默认)/column/row-reverse/column-reverse
  2. flex-wrap 换行 : nowrap(不换行，默认)/wrap
  3. flex-flow 1和2的简写
  4. justify-content 主轴对齐方式 : flex-start(往起点靠，默认)/flex-end/space-between(空间均匀放在中间)/space-around(空间均匀环绕)/center
  5. align-items 侧轴对齐方式 : stretch(伸展，所有元素伸展与宽或高相同，默认)/flex-start(起点对齐，不会伸展)/flex-end/center
  6. align-content 多行/列内容对齐方式(行/列直接的空隙)(使用较少) : stretch/flex-start(往起点靠，默认)/flex-end/space-between(空间均匀放在中间)/space-around(空间均匀环绕)
  • flex item属性
  1. flex-grow 增长比例(空间过多时，按何比例分配空间)
  2. flex-shrink 收缩比例(空间不够时，按何比例分配空间，一般不用)
  3. flex-basis 默认大小(一般不用，因为一般默认大小就是原始的大小)(注意不是basic)
  4. flex 1、2、3缩写
  5. order 顺序(代替双飞翼)(IE8不支持) : 比如order:1，可以改变展示顺序，使用时所有items都要进行设置
  6. align-self 自身对齐方式 : 不写的话对齐方式根据flex container
