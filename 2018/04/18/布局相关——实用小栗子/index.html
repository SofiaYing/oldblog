布局实用小栗子

块级元素垂直居中对齐方式——绝对定位元素margin:auto居中

margin:auto填充规则：

  • 如果一侧定值，一侧auto，则auto为剩余空间大小
  • 如果两次均为auto，则平均分配剩余空间

触发margin:auto计算有一个前提条件：width或height为auto时，元素具有对应方向的自动填充特性，所以正常使用时只能实现水平方向居中。

举例：son元素水平和垂直方向同时居中
`
.father{
position:relative;
width:300px;height:150px;
}
.son{
position:absolute; /此时son元素尺寸表现为“格式化宽度和格式化高度”，尺寸自动填充父级元素的可用尺寸/
top:o;right:0;bottom:0;right:0;
width:200px;height:100px;
margin:auto;
/由于绝对定位元素的格式化高度，即使父元素height:auto也同样支持，IE8及以上版本支持/
/对于替换元素，设置display:block，则计算规则相同/
}
